# T012 Scope Boundary Watchlist

## 层级

产品边界观察项。

## 状态

非近期实现项。除非产品边界被重新决策，否则不要实现。

## 背景

当前仓库明确只支持 React 与 Vue 新项目脚手架。Node 项目脚手架、远程模板、插件化模板来源和已有项目增量更新都不在当前支持范围。

## 目标

记录这些边界项，防止它们以“小功能”的形式绕过架构和产品决策进入实现。

## 观察项

1. Node 项目脚手架。
2. 远程模板来源。
3. 插件系统或可插拔模板来源。
4. 对已有项目做增量式更新。

## 进入实现前必须满足

1. 用户侧系统架构更新支持范围。
2. 执行约束更新产品边界。
3. 验证矩阵新增对应验证策略。
4. 重新评估 preserved core 是否仍足够。
5. 重新评估 command output redaction、安全边界和路径边界。

## 非目标

- 不把这些观察项当成默认路线图。
- 不为了未来可能性提前引入插件抽象。
- 不把远程模板需求混入当前本地模板 registry。
