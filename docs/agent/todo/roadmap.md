## 使用规则

1. 本目录记录未来可能执行的工作，不代表已经进入当前 milestone。
2. 执行前必须重新读取对应约束文档，尤其是架构、workflow materialization、验证矩阵和测试组织约束。
3. 如果 TODO 涉及用户可见行为，执行时必须同步检查用户文档。
4. 如果 TODO 触及当前明确不支持的产品边界，必须先有新的产品决策，再进入实现。
